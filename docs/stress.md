# Stress Testing

## Monitoring

The application below is used to monitor the status of running stress tests in near real time. It provides many different views of metrics generated by the system in addition to data generated by workload processes. When the test completes various reports are available in order to identify potential problems.

![](../images/test-monitoring.jpg)

## Failure Simulations

The graph below describes the various states of a distributed system during a failure and recovery test. The test simulated a failure of 3 out of 6 nodes to determine how the failures affected an active workload.
